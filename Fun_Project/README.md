# Snake Game (Python Turtle)

## Description
This project is a classic Snake Game built using Python's Turtle graphics library. The objective is to control the snake, eat food, and grow in length while avoiding collisions with walls and itself. The game also includes a leaderboard system to track high scores.

## Dataset
- Source: Not applicable (Game-based project)
- Description: No dataset is used. Game logic is based on real-time user interaction.

## Steps Performed
1. Game Setup using Turtle graphics
2. Snake movement implementation
3. Food spawning logic
4. Collision detection (wall & self)
5. Score tracking and leaderboard system
6. Game restart and user input handling

## Results
- Fully functional snake game
- Real-time score updates
- Leaderboard displaying top scores
- Smooth gameplay with keyboard controls

## Tools Used
- Python
- Turtle Graphics
- JSON (for leaderboard storage)
- Random, Time, OS modules

## Conclusion
This project demonstrates the use of Python for building interactive games. It highlights concepts like event handling, game loops, and file handling for persistent data storage.
