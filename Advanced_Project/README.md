# Electric Vehicle Clustering Analysis

## Description
This project focuses on analyzing Electric Vehicle (EV) data using clustering techniques to identify patterns and group similar data points. It helps in understanding EV adoption trends and market segmentation.

## Dataset
- Source: Kaggle / Provided dataset
- Description: Dataset includes EV-related features such as sales, region, vehicle type, and adoption trends.

## Steps Performed
1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Scaling and Transformation
4. Clustering using K-Means
5. Visualization of clusters
6. Interpretation of cluster groups

## Results
- Identified distinct clusters of EV adoption patterns
- Segmented regions/markets based on EV growth
- Improved understanding of EV trends

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (K-Means Clustering)

## Conclusion
The project demonstrates how clustering can be used to uncover hidden patterns in EV data, helping stakeholders make data-driven decisions regarding market strategies.
