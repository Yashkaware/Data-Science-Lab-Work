# Premier League Data Analysis

## Description
This project analyzes Premier League football data from multiple seasons to extract insights such as team performance, win rates, goal trends, and conversion efficiency. It also generates visualizations for better understanding.

## Dataset
- Source: Kaggle (Premier League dataset)
- Description: Match-level data including teams, goals, results, shots, and seasons (2006–2018). Synthetic data is used if dataset is unavailable.

## Steps Performed
1. Data Loading and Preparation
2. Data Cleaning and Structuring
3. Exploratory Data Analysis (EDA)
4. Feature Engineering (team stats, win rates, etc.)
5. Data Visualization using Matplotlib
6. Insight Extraction

## Results
- Identified top-performing teams based on total points
- Calculated home vs away win rates
- Determined most successful teams (titles won)
- Analyzed goal trends across seasons
- Evaluated shot-to-goal conversion efficiency

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib

## Conclusion
The project provides valuable insights into team performance and match trends in the Premier League. It demonstrates how data analysis and visualization can be used to extract meaningful patterns from sports data.
